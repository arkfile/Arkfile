Parameters (matching Arkfile config):

Press the button to start. This will allocate 128 MB of memory and may take 10-60 seconds depending on your device.
This benchmark uses the same @noble/hashes argon2id implementation and parameters as Arkfile. The result reflects how long key derivation takes in the browser on this device. No network requests are made.
